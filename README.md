# TSP-Search-Algorithms-Diculescu_Darius-
Acest repository conține implementarea temei de casă pentru cursul de Inteligență Artificială, axată pe problema Traveling Salesman (TSP). Proiectul include implementarea și testarea a trei algoritmi de căutare: Breadth-First Search, Uniform Cost Search și A* Search.Pentru detalii suplimentare, consultați fișierul README.md din repository.
